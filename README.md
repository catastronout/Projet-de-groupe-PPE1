- Description
Ce projet analyse l'usage du mot "autonomie" dans différentes langues à travers le web. Il collecte des pages web, extrait le texte, compte les occurrences de termes spécifiques et génère des concordanciers (KWIC).
Langues étudiées : Bélarussien, Coréen, Truc

- Résultats
Tableau HTML

<Colonnes>
N° : numéro de ligne
URL : lien vers la page
Code HTTP : statut de la requête
Encodage : encodage détecté
Nb mots : nombre total de mots
Occurrences : nombre de fois où les mots apparaissent
Dump HTML : page HTML brute
Dump Text : texte extrait
Concordancier : lien vers le KWIC
Bigrammes : paires de mots fréquentes
Robots.txt : vérification du fichier robots

- Site
Notre site se compose de quatre sections principales : Projet, Langues, Contributeurs et Scripts. Il propose également un mode sombre.

Dans la section Projet, il est possible de consulter la chaîne de traitement du projet. En cliquant sur « Voir le détail des scripts → », l’utilisateur est redirigé vers la section Scripts.

La section Langues comprend trois langues : le biélorusse, le coréen et le turc. Pour chaque langue, deux options sont disponibles : Tableau et Analyse. En cliquant sur l’une de ces options, l’utilisateur peut accéder aux informations détaillées correspondantes.

Enfin, la section Contributeurs présente les membres du projet ainsi que nos images.
